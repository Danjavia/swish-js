CSS Swish

A lightweight JavaScript library for animating elements using pre-defined CSS transitions. Includes both native JavaScript and jQuery plugin versions.

Basic Example

Native JS

var elem = document.querySelector('#selector');
elem.swish('swish-fade', 1000);

jQuery

$('#selector').swish('swish-fade', 1000);

Data Attributes

HTML

<div id="selector" data-swish-transition="swish-fade" data-swish-duration="500"></div>

Native JS

var elem = document.querySelector('#selector');
elem.swish();

jQuery

$('#selector').swish();

More Methods

Native JS

elem.swishIn();
elem.swishOut();

jQuery

$('#selector').swishIn();
$('#selector').swishOut();

Default State

HTML

<div class="example" id="selector"></div>

CSS

.example {
    display: none;
}

Custom Transitions

CSS

.example-transition {
    transition: all 1s ease-in-out;
}

.example-transition.out {
    background: blue;
}

.example-transition.in {
    background: red;
}

Native JS

var elem = document.querySelector('#selector');
elem.swish('example-transition');

jQuery

$('#selector').swish('example-transition');

Configuration

Native JS

Swish({
    hiddenClass: 'out',
    visibleClass: 'in'
});

jQuery

$.Swish({
    hiddenClass: 'out',
    visibleClass: 'in'
});
